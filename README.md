# Calculator CLI

A simple command-line calculator built with Python.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero
- Input validation

## Technologies Used

- Python 3
- Git
- GitHub
- VS Code

## How to Run

```bash
python calculator.py
```
